## 二、产品概述

### 2.1 一句话描述
祖宗模拟器：万族争霸！开局只是草履虫，分化三大种族，进化天赋、争领地、战天灾，角逐地表最强霸主。纯选择答题模式，3分钟一局，上头复玩。

### 2.2 核心循环
```
开局选种族 → 自身进化(10题) → 种族冲突(6题) → 随机奇遇(1题) → 大型天灾(6题) → 结局结算
                                  ↑___________一轮走完___________↓
```

### 2.3 核心亮点
1. **层层递进答题**：固定题型板块，10题进化是核心养成，冲突/天灾考验决策，节奏清晰
2. **选择连锁影响**：前序进化属性悄悄改变后续收益，每题有实际意义
3. **形态可视化**：每轮完成，生物外观同步升级，视觉成长感拉满
4. **进度条实时反馈**：每道题可见领地占有率变化，正向反馈持续刺激
5. **多结局驱动复玩**：刷满分霸主结局、解锁全部形态，反复挑战不同选择

## 四、整体业务流程

### 4.1 全局流程图

```
[启动] → [开局引导(剧情+种族选择)] → [自身进化(10题)] → [种族冲突(6题)] → [随机奇遇(1题)] → [大型天灾(6题)] → [结局结算]
```

### 4.2 进度条公式

$$总进度 = 10\%（初始） + 进化加成 + 冲突加成 + 奇遇加成 + 天灾加成$$

**各阶段加成明细：**

| 阶段 | 优秀/进取 | 普通 | 劣质/失败 |
|------|-----------|------|-----------|
| 初始开局 | +10%（固定） | — | — |
| 自身进化 | +40% | +20% | +5% |
| 种族冲突 | +35% | +15% | +0% |
| 随机奇遇 | 常规+5% / 隐藏+5% | — | — |
| 大型天灾 | +10%（完美） | +5%（勉强） | -10%（失误） |

**极端值验证（一轮）：**
- 最高：10% + 40% + 35% + 5% + 10% = 110% → 封顶100%
- 最低：10% + 5% + 0% + 5% + (-10%) = 10%

**典型路线示例：**
- 全优秀路线：10+40+35+5+10 = 100% → 100% 地球霸主
- 全普通路线：10+20+15+15+5 = 65% → 一方豪强
- 普通偏下：10+20+15+5+5 = 55% → 勉强存续
- 全差路线：10+5+0+5-10 = 10% → 族群衰落

### 4.3 结局判定

| 进度区间 | 结局名称 | 展示形态 |
|----------|----------|----------|
| 90%~100% | 🏆 地球霸主 | 终极霸主形态 |
| 60%~89% | ⚔️ 一方豪强 | 高阶形态 |
| 30%~59% | 🌱 勉强存续 | 基础形态 |
| <30% | 💀 族群衰落 | 弱化形态，画面偏灰暗 |

## 五、功能模块详细需求

### 5.1 开局引导模块

**功能描述**：展示初始剧情，玩家选择种族，初始化进度条。

**剧情文案**：
> 远古海洋之中，无数草履虫繁衍生息。环境不断变化，族群开始分化，演化出三大分支，由你带领开启称霸之路。

**种族选择（三选一）**：

| 种族 | 名称 | 初始形态 | 两大路线 | 进化链 |
|------|------|----------|----------|--------|
| 海洋霸主 | 巨型邓氏鱼 | 头甲鱼（孱弱古鱼） | 防御流 / 强攻流 | 头甲鱼→初始全骨鱼→粒骨鱼→巨型邓氏鱼 |

4. 进度条初始化为10%
5. 进入第一轮自身进化

### 5.2 自身进化模块

**功能描述**：种族专属10道二选一进化题，判定演化路线归属，结算档位评价。

#### 5.2.1 前置规则

- 题量：10题/轮，二选一
- 题目递进顺序：体型→攻击器官→行动方式→生存习性→感官→体能→抗逆性→繁衍→心智→最终演化定位
- 每题选择对应一条路线（A或B），统计10题中同一路线选择数量判定档位

#### 5.2.2 档位判定逻辑

| 档位 | 判定标准 | 进度加成 | 阶段评价文案 |
|------|----------|----------|-------------|
| 优秀 | 同一路线 ≥8题 | +40% | 演化方向精准统一，躯体与能力全面跃升 |
| 普通 | 同一路线 4~7题 | +20% | 演化路线摇摆，各项能力均衡，无突出优势 |
| 劣质 | 同一路线 ≤3题 | +5% | 演化杂乱失衡，身体结构出现缺陷，整体实力下滑 |

**种族形态详述（供美术/AI生图参考）：**

**邓氏鱼进化链：**
- 头甲鱼：无颌、无牙、无成对附肢；全身覆骨甲，底栖滤食；嘴只是简单口盘，不能主动咬
- 初始全骨鱼：出现最早的颌骨；头+前半身厚重分节骨甲，后半身裸露；能咬、能主动捕食；体型0.5–2m
- 粒骨鱼：骨甲缩短变灵活、颈部关节发达；颌骨变强、出现刃状骨齿；身体流线化，中上层游泳；体型2–3m
- 巨型邓氏鱼：装甲极致、颌骨特化为"骨刃"；咬合力峰值约5000–7000N；体长3.5–6m、体重1–4吨；泥盆纪海洋顶级掠食者

**刃齿虎进化链：**
- 细齿兽(Miacis)：猫科共同祖先，体型小巧似鼬，犬齿短而普通，无剑齿，擅长爬树，小型猎物杂食
- 伪剑齿虎(Hoplophoneus)：早期剑齿型食肉动物，上颌犬齿明显延长，身体粗壮，延长牙齿穿刺猎物，陆地伏击
- 纤细刃齿虎(Smilodon gracilis)：刃齿虎早期种类，弯刀状长牙成型，下颌张开幅度极大，四肢矫健，捕猎中大型兽类
- 毁灭刃齿虎(Smilodon populator)：刃齿虎巅峰物种，体型壮硕，剑齿修长锋利弧度完美，颈部力量极强，长牙切割猎杀巨兽

**孔子鸟进化链：**
- 中华龙鸟(Sinosauropteryx)：名为"鸟"实为"龙"；陆栖骨重、前肢短小；丝状原生羽毛、不具羽轴、缺乏空气动力；功能为控温孵卵、视觉炫耀
- 赫氏近鸟龙(Anchiornis)：名为"龙"无限"近鸟"；骨骼轻量、前肢拉长、半月形腕骨；不对称飞羽、四翼形态覆全身；树栖滑翔、降落缓冲
- 始祖鸟(Archaeopteryx)：鸟类"始祖"承上启下；双翼锁形、镶嵌结构（齿、长尾、指爪）；现代不对称飞羽、后肢羽毛退化；摆脱地面、主动扑翼
- 圣贤孔子鸟(Confuciusornis)：借"圣人"之名彰"真鸟"之实；牙齿退化具角质喙、长尾骨愈合具尾综骨；重心前移、尾综骨直插轻盈尾羽；胸骨完善、主动振翅

### 5.3 种族冲突模块

**功能描述**：领地争夺战，6道二选一决策题，进取得分高则扩张成功。

#### 5.3.1 前置剧情

> 领地扩张，遭遇其他族群/天敌进犯，战火一触即发，你的选择将决定胜负与领地归属。

#### 5.3.2 计分规则

- 进取选项 +1分，保守选项 0分
- 总分≥4 → 优秀(+35%)
- 总分2~3 → 普通(+15%)
- 总分≤1 → 劣质(+0%)

#### 5.3.3 题库（6题，选项顺序打乱）

| 题号 | 场景 | 选项1 | 选项2 |
|------|------|-------|-------|
| 1 | 敌方大举压境 | 据地防守(0 保守) | 主动迎击(+1 进取) |
| 2 | 探知敌方补给薄弱，出现突袭机会 | 按兵不动(0 保守) | 出兵偷袭(+1 进取) |
| 3 | 部分族人陷入敌军包围 | 舍弃断后，保存主力(0 保守) | 全力营救(+1 进取) |
| 4 | 遇上弱小中立族群 | 直接驱逐，独占资源(0 保守) | 接纳入伙，壮大族群(+1 进取) |
| 5 | 两军战斗陷入僵持 | 拖延对峙，慢慢耗敌(0 保守) | 集中全力强攻(+1 进取) |
| 6 | 成功战胜对手，如何处置败方族群 | 驱赶流放，划清界限(0 保守) | 收服同化，扩充实力(+1 进取) |

> **注意**：实际展示时，进取/保守选项的左右位置需随机打乱，避免玩家形成"总选右边"的习惯。

#### 5.3.4 结算文案

| 档位 | 文案 |
|------|------|
| 优秀 | 族群勇猛善战，步步向外扩张，在领地争夺中所向披靡，整体实力大幅提升。 |
| 普通 | 行事犹豫不决，攻守摇摆不定，族群发展平稳，却难以抢占更多生存优势。 |
| 劣质 | 一味避战退让，接连错失发展良机，族群活动范围不断受限，生存压力加剧。 |

### 5.4 随机奇遇模块

**功能描述**：种族冲突结束后概率触发1道奇遇题，纯福利，含隐藏结局分支。

#### 5.4.1 触发规则

- 冲突结束后，触发奇遇
- 触发时从3类奇遇中随机选1类
- MVP版本：若选择隐藏选项，隐藏副本在打出结局选项后追加（选特定常规选项后追问是否深入），暂不设计道具系统

#### 5.4.2 奇遇1：彗星降临（星宇探索）

| 项目 | 内容 |
|------|------|
| 剧情文案 | 天际划过明亮彗星，群星运转轨迹清晰可见，仿佛指引着远方未知星域。 |
| 隐藏选项A | 带领族人围观：进度+5%，族群积累星象知识，视野与智慧小幅提升 |
| 常规选项B | 立刻远离躲避：进度+5%，平安无事 |
| 副本触发 | 选A后追问："彗星光芒笼罩族群，星轨纹路清晰浮现，是否追随星轨前行？" |
| 副本选项A | 接纳星界微光，追随星轨前行 → **隐藏结局：星衍族群** |
| 副本选项B | 心生畏惧，全员撤离 → 副本失败，不触发隐藏结局 |
| 隐藏结局立意 | 族群挣脱星球束缚，向往浩瀚星海，踏上星际探索之路 |

#### 5.4.3 奇遇2：神秘种子（变异入侵植物）
